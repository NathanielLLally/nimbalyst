## Installation

To install any component, use:

```bash
npx shadcn@latest add @aicanvas/<slug>
```

For example:
```bash
npx shadcn@latest add @aicanvas/ai-job-cards
npx shadcn@latest add @aicanvas/glass-dock
npx shadcn@latest add @aicanvas/cube-carousel
```
